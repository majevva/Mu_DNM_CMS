In case you want to update new version of launcher to your players,
first you need to put new launcher in "temp.zip" including the new "x64.dll" and "x86.dll" files (launcher exe name must be "NewLauncherVersion.exe"
second, you need to edit the file "L_Version.wvd" which is in the file "L_Version.zip", and put the coresponding version of the new launcher version in "temp.zip"
